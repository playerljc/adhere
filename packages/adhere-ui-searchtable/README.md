# 简介
&ensp;&ensp;一种查询(CURD)表格的通用模式(如果 UI 没有明确给出查询表格的 UI，就可以用这个默认模式)

- 提供渲染查询的 UI
- 暴露最小集的事件让子类进行重写

# ✨ 特性
- 支持 react(17.x)
- ant-design(4.x)
- 支持国际化
- 支持修改主题
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11

# 📦 安装
```javascript
npm install @baifendian/adhere-ui-searchtable --save
``` 

```javascript
yarn add @baifendian/adhere-ui-searchtable
```
