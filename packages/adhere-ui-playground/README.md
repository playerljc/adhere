# 简介
&ensp;&ensp;一个代码展示的组件

- CodeTabPanel
- PlayGround
- PlayGroundMulit

# ✨ 特性
- 支持 react(18.x)
- 支持国际化
- 支持修改主题
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11
