# 简介
&ensp;&ensp;对 ant-design 的<Model>进行封装

- 支持使用 open 方式打开
- 支持 Modal 原始方法
- 支持预定义 Alert、Confirm
- 不与变量进行绑定，开箱即用的功能

# ✨ 特性
- 支持 react(18.x)
- ant-design(5.x)
- 支持国际化
- 支持修改主题
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11

# 📦 安装
```javascript
npm install adhere-ui-messagedialog --save
``` 

```javascript
yarn add adhere-ui-messagedialog
```
