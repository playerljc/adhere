# 简介
&ensp;&ensp;对值的监控
- 支持 all - 每一个依赖项都改变了才执行回调
- 支持 race - 只要有一个依赖项改变了就执行回调缓存
- 支持函数调用的 memoized 在函数对象中使用符号来存储一些信息
- 支持表达式求职的 memoized 依赖项列表和函数

# ✨ 特性
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器
