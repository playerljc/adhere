# 简介
&ensp;&ensp;历史记录返回操作
- 如果历史栈中没有记录则返回主页，或者可以自定义
- 如果可以返回则进行返回

# ✨ 特性
- 支持国际化
- 支持修改主题
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11

# 📦 安装
```javascript
npm install adhere-ui-historyback --save
``` 

```javascript
yarn add adhere-ui-historyback
```
