# 简介
&ensp;&ensp;浏览器嗅探的库(选用https://github.com/mumuy/browser库来实现)
- 用来嗅探用户浏览器的型号，版本，内核等信息

# ✨ 特性
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11

# 📦 安装
```javascript
npm install @baifendian/adhere-util-browsersniff --save
``` 

```javascript
yarn add @baifendian/adhere-util-browsersniff
```
