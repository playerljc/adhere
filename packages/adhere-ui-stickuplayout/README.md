# 简介
&ensp;&ensp;滚动固定头额布局

# ✨ 特性
- 支持 react(18.x)
- 支持国际化
- 支持修改主题
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11

# 📦 安装
```javascript
npm install @baifendian/adhere-ui-stickuplayout --save
``` 

```javascript
yarn add @baifendian/adhere-ui-stickuplayout
```
