# 简介
&ensp;&ensp;全局的遮罩
- 可以遮罩任意元素
- 遮罩是无侵入的
- 可以自定义 loading 文本和透明度
- 可以取消

# ✨ 特性
- 支持国际化
- 支持修改主题
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11

# 📦 安装
```javascript
npm install adhere-ui-globalindicator --save
``` 

```javascript
yarn add adhere-ui-globalindicator
```
