# 简介
&ensp;&ensp;基于 antd-design 的 Table 的列头筛选

# ✨ 特性
- 支持 react(18.x)
- ant-design(5.x)
- 支持国际化
- 支持修改主题
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11

# 📦 安装
```javascript
npm install @baifendian/adhere-ui-tableheadsearch --save
``` 

```javascript
yarn add @baifendian/adhere-ui-tableheadsearch
```
