# 简介
&ensp;&ensp;antd的Form构建组件

# ✨ 特性
- 支持 react(18.x)
- ant-design(5.x)
- 支持国际化
- 支持修改主题
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11

# 📦 安装
```javascript
npm install adhere-ui-formitemcreator --save
``` 

```javascript
yarn add adhere-ui-formitemcreator
```
