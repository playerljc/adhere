# 简介
&ensp;&ensp;警告的提示(就是为了全局统一)

# ✨ 特性
- 支持 react(18.x)
- antd-mobile(5.x)
- 支持国际化
- 支持修改主题
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11

# 📦 安装
```javascript
npm install @baifendian/adhere-mobile-ui-prompt-warnprompt --save
``` 

```javascript
yarn add @baifendian/adhere-mobile-ui-prompt-warnprompt
```
