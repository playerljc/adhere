# 简介
&ensp;&ensp;对富文本进行样式隔离的包装

# ✨ 特性
- 支持 react(18.x)
- 支持国际化
- 支持修改主题
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11

# 📦 安装
```javascript
npm install @baifendian/adhere-ui-richtext-sandbox --save
``` 

```javascript
yarn add @baifendian/adhere-ui-richtext-sandbox
```
