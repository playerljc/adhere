# 简介
&ensp;&ensp;删除确认提示，删除的时候弹出提示，确认后在执行操作

# ✨ 特性
- 支持 react(18.x)
- ant-design(5.x)
- 支持国际化
- 支持修改主题
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11

# 📦 安装
```javascript
npm install adhere-ui-confirm-delconfirm --save
``` 

```javascript
yarn add adhere-ui-confirm-delconfirm
```
