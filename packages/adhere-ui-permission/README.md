# 简介
&ensp;&ensp;权限的组件
- 设置所有权限
- 传入权限(支持多个)
- 可以自定义无权限的 UI

# ✨ 特性
- 支持 react(18.x)
- 支持国际化
- 支持修改主题
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11

# 📦 安装
```javascript
npm install @baifendian/adhere-ui-permission --save
``` 

```javascript
yarn add @baifendian/adhere-ui-permission
```
